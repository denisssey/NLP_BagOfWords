# NLP: Bag of Words vs Word2Vec.
Практическое исследование классических и современных методов векторизации текста. Bag of Words реализован  с нуля, без использования готовых реализаций из библиотек.

---

## Что делает этот проект.
1. **Bag of Words** - собственная реализация. Класс `SimpleBagOfWords` написан вручную, без sklearn. 
Умеет:
* Предобрабатывать русский и английский текст (приведение к нижнему регистру, очистка через регулярные выражения).
* Строить словарь по корпусу текстов.
* Преобразовывать тексты в частотные векторы.

#### Демонстрирует ключевые ограничения BoW:
1. Порядок слов **не важен**.
2. Омонимия не различается.
---
2. **Word2Vec** - семантические эмбеддинги.
* Загружает преобученную русскоязычную модель `word2vec-ruscorpora-300` через Gensim.
* Если модель недоступна - автоматически переключается на синтетические эмбеддинги (работает оффлайн). 
* Демонстрирует векторную арифметику: `король - мужчина + женщина ≈ королева`.
* Находит семантически близкие слова для существительных, глаголов и прилагательных.

## Установка и запуск
```
# Клонировать репозиторий
git clone https://github.com/denisssey/NLP_BagOfWords
cd NLP_BagOfWords

# Установить зависимости
pip install numpy matplotlib scikit-learn gensim

# Запустить
python NLP-BoW.py
```

_**Примечание:** скрипт работает и без Gensim — если предобученная модель недоступна, для визуализации автоматически используются синтетические эмбеддинги._
